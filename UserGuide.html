FireSafe User Guide

Welcome to the FireSafe User Guide. Below are instructions to help you get started with the FireSafe application.

STEPS ON HOW TO USE THE FIRESAFE

Step 1: Access the FireSafe
- Open your web browser/application and navigate to FireSafe application page/Home Page. - Click "Learn More" button on the homepage or by visiting the Application Information Page.

Step 2: Log In
- You will be prompted to log in with demo credentials provided on the screen (e.g., Username:demo_user, Password:demo_password). - Enter the credentials and click "Log In" to access the application.

Step 3: Explore the Dashboard
- Once logged in, you will be directed to the Dashboard. The dashboard provides an overview of your facility's fire safety metrics including temperature, smoke levels and carbon monoxide data. - The dashboard also displays any active fire safety and summary of key data from connected sensors.

Step 4: Real-time Monitoring
In the Monitoring Section, you can view real-time data. - Select the location or area of the building to monitor. - Data such as temperaature, smoke levels etc will be displayed in real time. - Any metrics exceeding the threshold will be highlighted, and alerts will be generated.

Step 5: Set Up Alerts
To receive receive notifications, go to Alerts section. - Customize alert settings by defining specific threshold (high temperature or smoke levels). - Choose how you want to be notified: via email or in-app notifications. - Save your Settings.

Step 6: Analyze Predictive Data
Access the Predictive Analytics section from the navigation menu. - View risk assessment based on historical fire data. - Review predictive models that assess potential fire risks in various locations within the facility. - Customize reports based on the data set you're most interested in (e.g., specific rooms or time periods).

Step 7: Use Communication Tools
The communication feature allows you to contact team members during fire-related events. - Select the Communication Tools from the dashboard. - Choose the team members you want to contact. - Use the built-in messaging or voice to tool to send critical informatioin quickly.

Step 8: View Reports
Navigate to the Reports section to generate detailed fire reports. - Choose the type of report you want (e.g., historical data, incident reports). - Customize the report by selecting specific data ranges or locations. - Export the report in PDF or CSV format for sharing or compliance purposes.

Step 9: Log Out
Once you're done exploring FireSafe, click Log Out button in the top-right corner to end session.
